Análisis de Ventas y Rentabilidad de Cursos — Power BI
Descripción del Proyecto
Este proyecto consiste en el desarrollo de un dashboard interactivo en Power BI para analizar el desempeño comercial y la rentabilidad de una empresa de formación.
El objetivo es identificar oportunidades de mejora en ventas, márgenes de beneficio y desempeño de comerciales y profesores mediante análisis de datos.

Herramientas Utilizadas
- Power BI
- DAX
- Modelado de datos
- Power Query
- Excel

KPIs Analizados
- Ventas Totales
- Beneficio Total
- Margen de Beneficio %
- Clientes Totales
- Ticket Promedio
- Ventas por País
- Rentabilidad por Curso
- Performance Comercial

Funcionalidades del Dashboard
- Segmentación por año
- Análisis temporal de ventas y beneficio
- Ranking de cursos más rentables
- Análisis geográfico
- Desempeño de comerciales y profesores

Insights de Negocio
El análisis permite identificar:
- Cursos con mayor rentabilidad
- Países con mejores resultados comerciales
- Comerciales con mejor desempeño
- Tendencias de ventas en el tiempo

Dataset
Datos simulados de una empresa de formación para fines educativos y de portfolio.
